Лабораторна робота №1
"Розробка статичного інтерфейсу Web-додатка"


Виконала:

Студентка групи КВ-31 
Верещінська Валерія 

Завдання на лабораторну роботу:

Загальне завдання: розробити статичні Web-сторінки додатку та перевірити коректне їх відображення на екранах різного розміру.

Інструменти розробки: мови HTML5, CSS3, фреймворки Twitter Bootstrap або Material UI

Деталізоване завдання:  Запропонувати тематику Web-додатку

1. Ознайомившись з переліком обов’язкових сторінок, виконати їх кодування у вигляді HTML-сторінок:

a. Встановити середовище розробки (рекомендовано VS Code)

b. Підключити на вибір фреймворк Twitter Bootstrap або Material UI

c. Розробити та відлагодити файли HTML та CSS (за необхідності)

2. За бажанням студента доповнити сайт іншими сторінками.

3. Створити репозиторій та завантажити усі файли на github.com.

4. Перевірити сторінки сайту на адаптивність до відображення на різних розмірах екрану (скористатись панеллю відлагодження Google Chrome або Mozilla Firefox).
